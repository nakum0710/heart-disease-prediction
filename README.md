# ❤️ Heart Disease Prediction Web App

A Machine Learning web application built using Streamlit that predicts heart disease risk using medical data.

---

# Features

- Interactive UI using Streamlit
- Real-time prediction
- Data preprocessing and scaling
- Machine Learning integration

---

# Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

---

# Run Locally

Install libraries:
```bash
pip install -r reuirements.txt
```
Run app:
```bash
streamlit run app.py
```
